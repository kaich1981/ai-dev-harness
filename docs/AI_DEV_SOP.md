# 🧠 AI Development SOP (Harness Engineering)

---

# 🇺🇸 English

## Core Philosophy

AI is not the developer. You are.

```
Prompt → Code → Chaos ❌
System → Contract → Task → Control ✅
```

---

## Workflow

```
Idea → Requirement → Contract → Task → AI → Validation → Iteration
```

---

## Key Rules

* No contract, no code
* One task at a time
* Every step must be verifiable
* AI only executes tasks

---

## Task Flow

1. Define requirement
2. Define contracts
3. Break tasks
4. Execute with AI
5. Validate
6. Commit
