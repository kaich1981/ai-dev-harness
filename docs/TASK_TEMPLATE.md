# 🧩 Task Template

---

# 🇺🇸 English

```markdown
# Task: <Name>

## Background
Describe problem

## Inputs
- Requirement
- API Contract
- DB Schema

## Outputs
- File paths
- Features

## Constraints
- No unrelated changes
- Follow contract

## Acceptance
- API works
- JSON correct
```
